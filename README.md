# Bikra Abna Portfolio

This is the personal portfolio website of **Bikra Abna**.

It showcases projects, experience, tech stack, and interactive UI sections with smooth animation and modern Web3-style visuals.

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **GSAP + ScrollTrigger** (`gsap`, `@gsap/react`)
- **React Icons**
